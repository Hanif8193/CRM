## Mission

> "Make every customer feel like your only customer."

Flowdesk helps small and mid-sized businesses deliver enterprise-grade customer support without enterprise-grade complexity. We believe great customer experience shouldn't require a team of 50 people to pull off.

---

## Product

**Flowdesk** is an AI-powered customer support platform that unifies all customer conversations — email, live chat, WhatsApp, and web forms — into a single shared inbox. It combines a human agent workspace with an AI assistant that can handle Tier-1 issues automatically, 24/7.

### Core Product Modules

| Module | Description |
|--------|-------------|
| **Unified Inbox** | All channels (email, chat, WhatsApp) in one place |
| **AI Agent** | Handles FAQs, onboarding questions, and common issues automatically |
| **Ticket Management** | Create, assign, prioritize, and track support tickets |
| **Knowledge Base** | Self-serve help center with search |
| **Analytics Dashboard** | Response times, resolution rates, CSAT scores |
| **Integrations** | Connects with Stripe, HubSpot, Slack, Zapier, and 40+ tools |

## Key Differentiators

1. **AI-first design** — The AI agent is not a bolt-on. It's built into the core workflow.
2. **5-minute setup** — Connect Gmail or WhatsApp and go live same day.
3. **No per-ticket pricing** — Flat monthly fee, no surprise bills.
4. **GDPR & SOC 2 Type II compliant** — Enterprise-grade security for all plans.
5. **Human handoff** — Seamless escalation from AI to human agents with full context.
